# **App Name**: Believers Network

## Core Features:

- Hero Section: Display the tagline: 'Transforming Lives. Impacting Nations. Spreading the Gospel' overlaid on a hero section with background video/image of youth worship and evangelism scenes.
- Animated CTAs: Implement glowing, scroll-animated CTAs like ‘Explore What We Do,’ ‘Meet the Vision Bearer,’ and ‘Join the Mission’ to encourage user interaction and spiritual exploration.
- Live Stat Counter: Include a live stat counter for metrics such as countries reached, students impacted, souls won, and gospel resources distributed to showcase the network’s impact.
- Personalized Devotionals: Daily devotionals: Each day, a new devotional is published. An AI tool personalizes devotional content for the reader, adapting tone and topic according to general spiritual maturity level.
- Responsive Design: Responsive Design: Ensure all website features adapt seamlessly to different screen sizes, providing an optimal viewing experience on desktops, tablets, and smartphones.
- Mobile Navigation: Mobile Navigation: Implement a mobile-first design featuring a responsive navigation bar and collapsible menu, improving usability on smaller screens.
- Footer Contact Information: Footer Information: The footer must include essential contact details such as phone numbers (+256752948431, +256766754286), email (hello@believersnetwork.com), a contact form, and social media icons for easy access and communication.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) evokes a sense of divinity and serenity.
- Background color: Desaturated sky blue (#E1F5FE) provides a light, calming backdrop.
- Accent color: Emerald green (#50C878) symbolizes growth and new life. Used for highlights and interactive elements.
- Headline font: 'Playfair', a modern serif for an elegant and high-end feel.
- Body font: 'PT Sans', a clean sans-serif for readability and a modern touch.
- Use spiritual icons that resonate with faith and campus revivals, creating a visual connection with the ministry’s mission and vision.
- Apply smooth transitions and animated counters (Framer Motion or AOS) to enhance user engagement and visual appeal. Ensure every transition feels faith-filled.
